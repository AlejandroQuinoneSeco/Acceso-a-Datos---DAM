# 📚 Acceso a Datos — DAM

Ejercicios, prácticas y exámenes de la asignatura **Acceso a Datos** del Ciclo Superior de Desarrollo de Aplicaciones Multiplataforma (DAM).

## 🗂️ Estructura

- **JDBC/** — Conexión a bases de datos relacionales con Java y JDBC
- **Hibernate_ORM/** — Mapeo objeto-relacional con Hibernate y JPA
- **NoSQL_MongoDB/** — Bases de datos NoSQL con MongoDB
- **Examenes/** — Exámenes de convocatoria ordinaria y extraordinaria

## 🛠️ Tecnologías

Java · JDBC · Hibernate · JPA · MongoDB · MySQL · XML
